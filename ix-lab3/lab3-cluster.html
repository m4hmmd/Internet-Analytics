# distance function based on Jaccard index
def d_j(a, b):
    return 1 - len(movie_genres[a] & movie_genres[b])/len(movie_genres[a] | movie_genres[b])

def KMedioids(movie_ids, k=2):
    # select centers out of all movies, without replacement
    centers = random.sample(movie_ids, k)
    print(centers)
    n_iter = 7 # it converges very fast, that's why we use only a few iterations
    for t in range(n_iter):
        clusters = [set() for j in range(k)]
        
        # Assign points to clusters
        for movieId in movie_ids:
            i = int(np.argmin([d_j(center, movieId) for center in centers]))
            clusters[i].add(movieId)
        print('updating centers')
        # Update the centers
        for i in range(k):
            centers[i] = list(clusters[i])[0]
            minSum = sum([d_j(centers[i], y) for y in clusters[i]])
            for x in clusters[i]:
                if sum([d_j(x, y) for y in clusters[i]]) < minSum:
                    centers[i] = x
        print("centers after iter {}{}:".format(t, centers))
    return [0 if movieId in clusters[0] else 1 for movieId in movie_ids]

It's very difficult to interpret the two clusters, espacially because of the insufficiency of 1. our way of visualizing the clustering, 2. the size of the range of our distance function (it can only take a limited number of values in this dataset) and 3. suboptimality of the convergence of k-medioids algorithm.

We will give it a try though. The above graph shows that Drama, Romance, Crime, Thriller, Mystery, War and Film-Noir movies tend to cluster together in cluster 1 (in blue). This may be attributeable to the fact that {Drama, Romance} and {Crime, War, Mystery, Thriller} genres often appear together in a list of genres of most movies. Also, {Adventure, Comedy, Children, Fantasy, SciFi} aldo tend to appear together, explaining why they're in the same cluster, while for example 'Mystery' rarely ever appears along 'Children' and 'Comedy' rarely appearl along with 'Drama' (in other words, there aren't many movies which have both genres Drama and Comedy).

If we were to categorically name the groups, we would probably say the 'blue' cluster contains movies more wathced by adults, and 'red' cluster has movies that can be watched by viewers of any age, but are preferred more by younger audience.
